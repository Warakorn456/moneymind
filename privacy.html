นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 24 มิถุนายน 2569 (2026)

MoneyMind ("เรา") ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้ทุกคน นโยบายนี้อธิบายว่าเราเก็บรวบรวม ใช้ และปกป้องข้อมูลของคุณอย่างไร เมื่อคุณใช้งานแอปพลิเคชัน MoneyMind ทั้งบนเว็บและอุปกรณ์มือถือ

1. ข้อมูลที่เราเก็บรวบรวม

MoneyMind เก็บข้อมูลที่คุณป้อนเองเท่านั้น ได้แก่:

  • ข้อมูลบัญชีผู้ใช้ (ชื่อผู้ใช้ อีเมล รหัสผ่านในรูปแบบที่เข้ารหัสแล้ว)
  • ข้อมูลทางการเงิน เช่น รายรับ รายจ่าย เป้าหมายเงินออม หนี้สิน และพอร์ตลงทุน
  • การตั้งค่าและค่ากำหนดส่วนตัวภายในแอป

เราไม่เก็บข้อมูลที่ระบุตัวตนได้โดยอัตโนมัติ เช่น ตำแหน่งที่ตั้ง, Cookies การติดตาม, หรือพฤติกรรมการใช้งาน

2. วิธีที่เราใช้ข้อมูลของคุณ

ข้อมูลของคุณถูกใช้เพื่อ:

  • แสดงผลข้อมูลทางการเงินส่วนตัวของคุณเท่านั้น
  • ซิงค์ข้อมูลข้ามอุปกรณ์ผ่าน Firebase Firestore (เฉพาะบัญชีของคุณ)
  • ให้บริการฟีเจอร์ AI วิเคราะห์ข้อมูลการเงิน (ผ่าน Google Gemini API)

เราไม่ขาย ไม่เช่า และไม่แบ่งปันข้อมูลส่วนตัวของคุณแก่บุคคลที่สามเพื่อวัตถุประสงค์ทางการตลาด

3. การจัดเก็บข้อมูล

  • ในอุปกรณ์: ข้อมูลถูกจัดเก็บใน Local Storage ของอุปกรณ์คุณ
  • บนคลาวด์: ข้อมูลซิงค์ไปยัง Google Firebase Firestore ในบัญชีที่ปลอดภัยของคุณ (ต้องล็อกอินก่อนเข้าถึง)
  • ข้อมูลของคุณถูกป้องกันด้วย Firebase Security Rules — ผู้อื่นเข้าถึงข้อมูลของคุณไม่ได้

4. ความปลอดภัยของข้อมูล

เราใช้มาตรการรักษาความปลอดภัย ดังนี้:

  • รหัสผ่านถูก hash ด้วย PBKDF2 ก่อนจัดเก็บ — ไม่มีใครเห็นรหัสผ่านจริงของคุณ
  • การเชื่อมต่อทุกอย่างใช้ HTTPS
  • Firebase Authentication ปกป้องการเข้าถึงข้อมูลบนคลาวด์

5. บริการของบุคคลที่สาม

MoneyMind ใช้บริการภายนอกดังนี้:

  • Google Firebase — จัดเก็บข้อมูลบนคลาวด์และการยืนยันตัวตน
  • Google Gemini API — วิเคราะห์ข้อมูลการเงินด้วย AI (เฉพาะเมื่อคุณเปิดใช้งาน)
  • CoinGecko API — ดึงราคาคริปโตเคอร์เรนซีแบบเรียลไทม์

บริการเหล่านี้มีนโยบายความเป็นส่วนตัวของตัวเองซึ่งคุณสามารถตรวจสอบได้จากเว็บไซต์ของแต่ละบริษัท

6. สิทธิ์ของคุณ

คุณมีสิทธิ์:

  • ดู แก้ไข และลบข้อมูลทั้งหมดของคุณได้ตลอดเวลาผ่านแอป
  • ส่งออกข้อมูลของคุณในรูปแบบ CSV
  • ลบบัญชีและข้อมูลทั้งหมดได้ตามต้องการ

7. ข้อมูลของเด็ก

MoneyMind ไม่ได้ออกแบบมาสำหรับเด็กอายุต่ำกว่า 13 ปี และไม่ตั้งใจเก็บข้อมูลของเด็กโดยเฉพาะ

8. การเปลี่ยนแปลงนโยบาย

หากเราปรับปรุงนโยบายความเป็นส่วนตัว เราจะอัปเดตวันที่ "อัปเดตล่าสุด" ที่ด้านบนของหน้านี้ การใช้งานต่อเนื่องหลังจากการเปลี่ยนแปลงถือว่าคุณยอมรับนโยบายใหม่
